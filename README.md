![Timesy](./public/ogp.png)

**Timesy**は開発者のためのソーシャル分報アプリです。

## Requirement

- Docker
- Docker Compose

## Development

```sh
gh repo clone moekidev/timesy
```

```sh
cd timesy
```

```sh
docker compose run --rm web bin/setup
```

```sh
docker compose up
```
